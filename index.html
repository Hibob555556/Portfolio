QuickQRForge - URL-to-QR-code generator (Python)

A small but fully-featured Python application I built to make creating QR codes from URLs painless.

  • Dual interfaces - either run it interactively with a text-based UI (TUI) or invoke the command-line tool (qrgen) from scripts.
  • Smart URL handling - normalises addresses (adds https:// when missing) and validates input, returning meaningful exit codes for easy automation.
  • PNG output with configurable scale, suitable for embedding in websites, documents or stickers.
  • Reusable library - exposes a simple public API (quickqrforge.generate_qr) and the same functionality can be driven via subprocess for existing codebases.
  • Quality-of-life features - packaged for PyPI, installable with pip install QuickQRForge; includes comprehensive tests, GitHub Actions CI workflow, and clean MIT-licensed source.
  • Dev-friendly - supports editable installs, documented usage/examples, and a minimal dependency list.

This project demonstrates my ability to design a user-friendly CLI/TUI tool, handle input validation, write automated tests, and manage packaging/distribution for Python applications.

WebScraping - Automated Web Data Extraction (C# / Playwright)

A modern web scraping and automation tool built in C# using Playwright to reliably extract data from dynamic websites.

  • Browser automation - uses Playwright for fast, reliable interaction with modern JavaScript-heavy sites (no brittle HTML-only scraping).
  • Async-first design - leverages async/await for efficient, non-blocking scraping workflows.
  • Dynamic content handling - supports waiting for selectors, navigation events, and fully rendered DOM content.
  • Reusable architecture - modular scraping utilities designed for extension into larger data pipelines or automation systems.
  • Real-world use case - demonstrates practical experience with web automation, data extraction, and handling edge cases like timeouts and element states.

This project highlights my experience with web automation frameworks, asynchronous programming in C#, and building reliable tools for interacting with real-world web applications.
