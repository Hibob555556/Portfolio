QuickQRForge - URL-to-QR-code generator (Python)

A small but fully-featured Python application I built to make creating QR codes from URLs painless.

  • Dual interfaces - either run it interactively with a text-based UI (TUI) or invoke the command-line tool (qrgen) from scripts.
  • Smart URL handling - normalises addresses (adds https:// when missing) and validates input, returning meaningful exit codes for easy automation.
  • PNG output with configurable scale, suitable for embedding in websites, documents or stickers.
  • Reusable library - exposes a simple public API (quickqrforge.generate_qr) and the same functionality can be driven via subprocess for existing codebases.
  • Quality-of-life features - packaged for PyPI, installable with pip install QuickQRForge; includes comprehensive tests, GitHub Actions CI workflow, and clean MIT-licensed source.
  • Dev-friendly - supports editable installs, documented usage/examples, and a minimal dependency list.

This project demonstrates my ability to design a user-friendly CLI/TUI tool, handle input validation, write automated tests, and manage packaging/distribution for Python applications.

  • Real-world use case - demonstrates practical experience with web automation, data extraction, and handling edge cases like timeouts and element states.

This project highlights my experience with web automation frameworks, asynchronous programming in C#, and building reliable tools for interacting with real-world web applications.

SunTracker - Solar Elevation Calculator (C / WebAssembly)

A high-performance solar elevation calculator built in C and compiled to WebAssembly, with a JavaScript frontend for real-time visualization.

  • High-performance computation - core solar calculations implemented in C and compiled to WebAssembly for near-native performance in the browser.
  • Real-time elevation tracking - calculates current solar elevation based on user-provided latitude, longitude, altitude, and system time.
  • Full-day visualization - generates a chart of solar elevation throughout the day using Chart.js.
  • Clean UI/UX - responsive interface with dynamic panels that reveal results and graphs only when requested.
  • Systems + frontend integration - demonstrates bridging low-level C logic with modern JavaScript via Emscripten and cwrap.

This project highlights my ability to work across system-level programming and frontend development, optimize performance using WebAssembly, and build interactive, data-driven user interfaces.

Decimal & Binary Converter - WebAssembly Tool (C / WASM / JavaScript)

A high-performance number conversion tool built in C and compiled to WebAssembly, with a modern JavaScript UI for real-time interaction.

  • WebAssembly integration - core conversion logic written in C and compiled to WASM using Emscripten for near-native performance in the browser.
  • Efficient algorithms - implements decimal-to-binary and binary-to-decimal conversions with optimized logic and minimal overhead.
  • Formatted binary output - outputs padded binary grouped into 8-bit octets for improved readability.
  • Seamless JS ↔ C bridge - uses cwrap to connect frontend inputs to compiled C functions.
  • Custom UI - terminal-inspired responsive interface with real-time validation and dynamic updates.
  • Systems + frontend engineering - demonstrates combining low-level memory-managed code with modern web technologies.

This project highlights my ability to integrate low-level systems programming with frontend development, optimize browser performance using WebAssembly, and build clean, user-focused interfaces.
